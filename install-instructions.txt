instructions
============
1. make sure you have JDK installed
2. make sure you have $JAVA_HOME in PATH
3. install mvn
4. install psql
for example in debian:
sudo apt-get intall postgresql-9.6 postgresql-client-9.6
- note that the version 9.6 may alter
5. run in terminal
sudo su -u postgres psql postgres
6. use in postgreSQL UI 
create database "Dlearn_db";
7. use in postgreSQL UI
alter user postgres with password admin;
8. run from backend root
psql -U  postgres -W -h localhost -d "Dlearn_db" -f database/db.sql
9. install debendencies
mvn clean install
10. create jar file
mvn jar:jar
11. run the jar file
java -jar target/dependency/webapp-runner.jar target/*.war
